# Contributing to Top-AI-repos

A curated list of top AI repositories. Contributions welcome!

## How to Contribute

### Adding a Repository
1. Fork this repository
2. Add the repo to the appropriate section
3. Include: name, description, star count, and why it's notable
4. Keep entries alphabetical within sections
5. Open a Pull Request

### Guidelines
- Only add repos with significant community traction (100+ stars)
- Write concise, informative descriptions
- Place repos in the most relevant category
- Don't duplicate existing entries

### Reporting Issues
- Broken links
- Outdated information
- Missing categories

## Questions?

Open an issue on this repo.
